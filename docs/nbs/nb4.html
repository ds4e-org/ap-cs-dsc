Line charts should be used with summarized values (for example, mean, median, proportion), not individual responses. In Java, creating these summaries would typically require writing loops to group records, keep running totals, and compute a final value for each year. In Pandas, the .groupby() method provides a more efficient way to group data by a variable like survey_year and calculate the summary statistic we want to use for plotting.

.groupby()¶

The .groupby() method in Pandas separates data into groups based on the values of one or more variables. Rows that share the same value for a grouping variable (for example as survey year) are placed into the same group. Within each group, we compute summary statistics (for example the mean) to produce a single value per group.

4.0 - Comparing Multiple Trends Over Time¶

In this section, we use multi-line charts to compare how values change over time across groups.

Multi-Line Charts (Numerical Data)¶

A multi-line chart is useful when we want to examine change over time for multiple groups simultaneously. Each line represents a different group, making it possible to compare patterns and trends across categories within the same chart.

The first step in creating a multi-line chart is to choose a categorical variable to define the groups being compared. In this case, the variable age_group is useful because it organizes ages into broad categories. Creating a separate line for each individual age, or for groups that are too granular, would result in a cluttered chart that is hard to read.

After selecting, we create a group object that includes both age_group and survey_year. Here, we create the group object grps_age_groups:

In [17]:
## Run this code but do not edit it. Hit Ctrl+Enter to run the code
# Group the data by survey year and age group and store the result in grps_age_group
grps_age_groups = gss.groupby(["survey_year", "age_group"])
Instructor Note (Code):
  • gss.groupby(["survey_year", "age_group"]) groups the data simultaneously by survey year and age group. This creates groups for every unique combination of year and age group, allowing summary statistics (such as means or counts) to be computed separately for each group combination (e.g., 2002 for the age group 18-25 (Young Adult).
  • Using multiple grouping variables in a single .groupby() operation is the standard approach when preparing data for multi-line charts, since it produces one summarized value per group per time point.

Visual data moves involve transforming data into graphical representations to support interpretation, comparison, and questioning. Here, we focused on using line-based visualizations to explore change over time and differences across groups.

  • Line charts show how a variable changes across an ordered variable, such as time.
  • Multi-line charts allow us to compare how those changes differ across multiple groups within the same plot.

The visualizations created in this notebook demonstrate how line charts and multi-line charts support different data moves, helping us identify trends, compare groups, and generate new questions that may not be clear from tables or summary statistics alone.
